Privacy Policy

Terakhir diperbarui: April 2026

⚡ Singkatnya: Kami hanya menyimpan username dan foto profil Discord kamu. Kami tidak punya akses ke password, email, atau akun Discord kamu. Data kamu tidak dijual ke siapapun.

1. Data yang Kami Kumpulkan

Ketika kamu login menggunakan Discord, kami menyimpan informasi berikut:

  • Discord ID — angka unik yang mengidentifikasi akun Discord kamu
  • Username Discord — nama pengguna kamu di Discord
  • Foto profil — avatar Discord kamu (disimpan sebagai kode referensi, bukan file gambar)
  • Status paket — apakah akun kamu Free, Basic, atau Premium
  • Jumlah export — berapa kali kamu menggunakan fitur export hari ini

2. Data yang TIDAK Kami Kumpulkan

Kami tidak memiliki akses ke:

  • Password Discord kamu
  • Email kamu
  • Server Discord yang kamu ikuti
  • Pesan DM atau percakapan kamu
  • Informasi pembayaran apapun
  • File audio yang kamu proses — semua pemrosesan dilakukan di browser kamu sendiri

3. Bagaimana Login Discord Bekerja

Nova Audio menggunakan Discord OAuth2 untuk proses login. Cara kerjanya sama seperti "Login with Google" di website lain:

  • Kamu diarahkan ke halaman login resmi Discord
  • Kamu login langsung ke Discord — kami tidak pernah melihat password kamu
  • Discord memberikan kami "tiket" berisi username dan foto profil saja
  • Kami tidak bisa melakukan apapun atas nama kamu di Discord

Kamu bisa melihat sendiri scope yang diminta saat login — kami hanya meminta akses identify yang berarti hanya membaca profil publik kamu.

4. Bagaimana Data Disimpan

Data kamu disimpan di Supabase, layanan database yang aman dengan enkripsi standar industri. Session login disimpan sebagai cookie yang hanya bisa dibaca oleh server (HttpOnly), tidak bisa diakses oleh JavaScript atau pihak ketiga.

5. Berbagi Data

Kami tidak menjual, menyewakan, atau membagikan data kamu ke pihak ketiga manapun. Data kamu hanya digunakan untuk menjalankan layanan Nova Audio Studio.

6. Penghapusan Data

Jika kamu ingin data kamu dihapus dari sistem kami, hubungi kami melalui Discord server kami di discord.gg/saNDWMhVt9 dan kami akan menghapus data kamu dalam waktu 7 hari kerja.

7. Perubahan Kebijakan

Jika ada perubahan pada kebijakan privasi ini, kami akan mengumumkannya di server Discord kami. Dengan terus menggunakan Nova Audio, kamu menyetujui kebijakan yang berlaku.

8. Hubungi Kami

Ada pertanyaan soal privasi? Hubungi kami langsung di Discord server kami: discord.gg/saNDWMhVt9
